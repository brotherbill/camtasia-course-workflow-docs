# Camtasia Course Workflow Docs

This repository contains the deterministic, scalable documentation for the full Camtasia-based course production pipeline.  
It includes workflows, naming conventions, folder architecture, tool behaviors, and troubleshooting guides.

## Structure

- `workflows/` — Complete production workflows (minimal, full, hybrid, state-machine)
- `naming-conventions/` — Deterministic naming rules for lessons and assets
- `folder-architecture/` — Canonical 5-folder pipeline and variants
- `tools/` — Audiate and Camtasia behavior, settings, and integration notes
- `troubleshooting/` — Known issues and deterministic fixes
